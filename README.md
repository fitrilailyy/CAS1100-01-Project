This is a mini Python turtle program that associate average Internet usage of a country and its latest updated literacy rate.

This program consists of: -
- 1 welcome page (prompt user to input the .csv dataset files for look up)
- 1 main page with 4 options for the users to choose from:
  option 1. Look up specific countries
  option 2. Compute correlation
  option 3. Show top statistics
  option 4. Exit program.

To run this program, simply run "python3 project.py" OR "python project.py" OR if in VSCode, just simply click the run button.

Environment: Python 3.13.7 or similar ones.
